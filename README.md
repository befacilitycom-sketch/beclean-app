# beclean-app

BeClean Facility Group - Plateforme B2B de services de nettoyage professionnel et vente de produits d'hygiène au Maroc.
Application Web construite avec **Next.js 14 App Router**, **Payload CMS 3.0**, et **Serwist** (PWA).

## Fonctionnalités
- Intégration PWA complète (installable hors ligne).
- Design Premium (Glassmorphism, animations fluides).
- Devis en ligne avec gestion de soumission via Nodemailer.
- Interface d'Administration intégrée (Payload CMS).
- Déploiement multi-environnement optimisé (Docker).
